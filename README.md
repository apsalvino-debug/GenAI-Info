# My GenAi website

## Overview
This is my website project for Fundamentals of Computing. It includes informations about index, FAQ, ethics, article and with styling using CSS.

## How to Run
1. Clone the repository or download it as a ZIP.
2. Open `index.html` in a web browser.

## Frameworks/Libraries Used
- **Bootstrap 5.3.2** – for responsive layout and styling.
- **Google Fonts** – Roboto and Georgia for typography.
- **Custom CSS** – style.css for additional styling and animations.
